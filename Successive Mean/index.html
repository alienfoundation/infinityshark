About MEAN Stack

MEAN.JS is full stack JavaScript solution that facilitates the creation of fast, robust and maintainable production web applications. MEAN is an acronym for the four main technologies:

Mongo DB

MongoDB features a document database that is scalable & flexible. It involves dynamic querying, indexing and real-time aggregation that renders compelling methods to analyze your data. Its document model enables you to work with data easily.

Express

Express is a flexible Node.js web application framework comprising a thin layer of fundamental web app features without obscuring Node.js' infamous attributes.

Angular JS

AngularJS is an open source front end web application framework based on JavaScript which allows you to extend and expand the HTML vocabulary. It serves as a platform for the development of web applications.

Node JS

Node is designed to build scalable network applications. It is a cross-platform JS based framework and is used to develop I/O intensive web applications. It is lightweight and efficient, perfect for data-intensive real-time applications.
